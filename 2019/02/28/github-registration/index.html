总结

注册账号属于基本操作，没有什么太大的难度。唯一的障碍，可能就是GitHub的官网都是英文的。这里我们可以使用谷歌浏览器提供的“翻译”功能，把网页翻译成中文的。如此，我们就能看得更加明白了。
在上面第三步操作里的两张截图，第一张是在线翻译后的，第二张是英文版的原网页，翻译还是很准确的。我记得上大学的时候，都六七年前了，那时候用过几次“在线翻译”的功能，用了几次效果都不理想，翻译出来的结果都很是不准确，所以从那以后，就再没有用过这个功能了。
